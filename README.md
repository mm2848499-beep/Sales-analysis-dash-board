# 📊 Sales Performance & Interactive Analysis Dashboard

## 📌 Project Overview
This project provides a comprehensive analysis of sales data to identify high-performing regions, analyze product trends, and monitor Key Performance Indicators (KPIs) to support data-driven business decisions.

---

## 🛠 Tools Used
* **Microsoft Excel:** Advanced formulas, Data Cleaning, and Pivot Tables.
* **Data Visualization:** Interactive Dashboard with Slicers for dynamic reporting.

---

## 📂 Project Steps
1. **Data Cleaning:** Processed raw sales records to ensure data integrity and accuracy.
2. **Data Modeling:** Utilized **Pivot Tables** to aggregate data by Region and Product categories.
3. **KPI Calculation:** Extracted critical metrics including Total Sales ($2.6M+), Total Quantity, and Average Transaction Value.
4. **Interactive Dashboard:** Designed a user-friendly interface linking visual charts with **Slicers** for real-time filtering.

---

## 📊 Business Questions Addressed
* Which **Region** contributes the most to total revenue?
* How are sales distributed across different **Product Categories** (Laptops, Tablets, etc.)?
* What is the **Average Sales** value per transaction?

## 🔍 Key Business Insights
* **Regional Dominance:** The **North** and **West** regions are the top performers, accounting for approximately **52%** of total sales.
* **Product Performance:** While **Mice** and **Tablets** lead in sales volume (Quantity), **Laptops** drive the highest revenue value.
* **Operational Efficiency:** The system successfully tracks over **$2.6 Million** in total sales with clear visibility into average performance per region.
